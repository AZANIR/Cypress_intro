# Практичні вправи - Тема 8: Cypress Cloud

## Вправа 1: Налаштування проекту

**Завдання:**
1. Зареєструйтеся на Cypress Cloud
2. Створіть новий проект
3. Додайте Project ID в `cypress.config.js`
4. Запустіть тести з `--record` флагом

**Підказки:**
- Використовуйте змінні оточення для Record Key
- Не комітьте ключі в репозиторій

**Рішення:** Дивіться в `setup-guide.md`

---

## Вправа 2: Перегляд результатів

**Завдання:**
1. Запустіть тести з записом
2. Перейдіть в Cypress Cloud Dashboard
3. Перегляньте результати виконання
4. Перегляньте відео та скріншоти

**Підказки:**
- Всі запуски зберігаються в Cloud
- Можна порівняти різні запуски

**Рішення:** Практична вправа - виконується вручну

---

## Вправа 3: CI/CD інтеграція (опціонально)

**Завдання:**
Якщо у вас є доступ до CI/CD:
1. Налаштуйте GitHub Actions або інший CI/CD
2. Додайте Record Key як секрет
3. Налаштуйте автоматичний запуск тестів

**Підказки:**
- Дивіться `ci-integration.md` для прикладів
- Використовуйте секрети для безпеки

**Рішення:** Дивіться в `ci-integration.md`
