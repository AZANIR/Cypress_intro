# Практичні вправи - Тема 1: Огляд Cypress

## Вправа 1: Створення першого тесту

**Завдання:**
Створіть тест, який:
1. Відкриває TodoMVC
2. Перевіряє, що заголовок містить текст "todos"
3. Перевіряє, що поле вводу видиме

**Підказки:**
- Використовуйте `cy.visit('/')` для відкриття сторінки
- Використовуйте `cy.get('h1')` для пошуку заголовка
- Використовуйте `.should('contain', 'todos')` для перевірки тексту

**Рішення:** Дивіться в папці `solutions/`

## Вправа 2: Порівняння E2E та компонентних тестів

**Підказки:**

## Вправа 3: Огляд функціоналу TodoMVC

**Завдання:**
Створіть тест, який перевіряє наявність основних елементів:
1. Заголовок сторінки
2. Поле для введення задач
3. Фільтри (All, Active, Completed)
4. Лічильник задач

**Підказки:**
- Використовуйте `cy.get()` для пошуку елементів
- Використовуйте `.should('be.visible')` для перевірки видимості

**Рішення:** Дивіться в папці `solutions/`
