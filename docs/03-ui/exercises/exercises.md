# Практичні вправи - Тема 3: Cypress UI та Test Runner

## Вправа 1: Використання debug()

**Завдання:**
Створіть тест, який:
1. Відкриває TodoMVC
2. Додає задачу
3. Використовує `cy.debug()` перед перевіркою
4. Продовжує виконання після паузи

**Підказки:**
- Використовуйте `cy.debug()` для зупинки виконання
- Натисніть "Resume" в Command Log для продовження

**Рішення:** Дивіться в папці `solutions/`

---

## Вправа 2: Аналіз Command Log

**Завдання:**
1. Запустіть будь-який тест в UI режимі
2. Клікніть на кожну команду в Command Log
3. Перегляньте снепшоти для кожної команди
4. Проаналізуйте стан DOM на різних етапах

**Підказки:**
- Кожна команда має свій снепшот
- Використовуйте стрілки для навігації між снепшотами

**Рішення:** Практична вправа - виконується вручну

---

## Вправа 3: Використання pause()

**Завдання:**
Створіть тест з `cy.pause()` та:
1. Виконайте команди вручну після паузи
2. Перевірте стан сторінки
3. Продовжіть виконання тесту

**Підказки:**
- `cy.pause()` дозволяє інтерактивне дослідження
- Можна виконувати команди вручну в Command Log

**Рішення:** Дивіться в папці `solutions/`
