# Практичні вправи - Тема 2: Встановлення та конфігурація

## Вправа 1: Налаштування baseUrl

**Завдання:**
1. Відкрийте `cypress.config.js`
2. Додайте `baseUrl` для TodoMVC
3. Створіть тест, який використовує відносний шлях `cy.visit('/')`

**Підказки:**
- baseUrl має бути: `'https://todomvc.com/examples/react/dist/'`
- Після налаштування можна використовувати `cy.visit('/')` замість повного URL

**Рішення:** Дивіться в папці `solutions/`

---

## Вправа 2: Створення першого тесту

**Завдання:**
Створіть файл `cypress/e2e/my-first-test.cy.js` з тестом, який:
1. Відкриває TodoMVC
2. Додає задачу
3. Перевіряє, що задача додалася

**Підказки:**
- Використовуйте структуру `describe()` та `it()`
- Використовуйте `cy.get('.new-todo').type('...{enter}')`

**Рішення:** Дивіться в папці `solutions/`

---

## Вправа 3: Перевірка встановлення

**Завдання:**
1. Запустіть `npx cypress verify`
2. Перевірте, що все встановлено правильно
3. Запустіть `npx cypress open` та перевірте UI

**Підказки:**
- Якщо є помилки, перевірте версію Node.js
- Переконайтеся, що Cypress встановлено в `devDependencies`

**Рішення:** Перевірте вивід команд у терміналі
