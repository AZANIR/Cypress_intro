# Практичні вправи - Тема 5: Взаємодія з елементами

## Вправа 1: Пошук елементів

**Завдання:**
Створіть тест, який знаходить елементи різними способами:
1. За класом (`.new-todo`)
2. За тегом (`h1`)
3. За атрибутом (`[type="text"]`)
4. За текстом (`cy.contains('todos')`)

**Підказки:**
- Використовуйте різні типи селекторів
- Перевіряйте, що елементи знайдені

**Рішення:** Дивіться в папці `solutions/`

## Вправа 2: Взаємодія з елементами

**Завдання:**
Створіть тест, який:
1. Вводить текст у поле (`cy.type()`)
2. Клікає на чекбокс (`cy.check()`)
3. Клікає на кнопку фільтра (`cy.click()`)
4. Очищає поле (`cy.clear()`)

**Підказки:**
- Використовуйте різні команди взаємодії
- Перевіряйте результати кожної дії

**Рішення:** Дивіться в папці `solutions/`

## Вправа 3: Повний workflow

**Завдання:**
Створіть тест, який виконує повний цикл:
1. Додає 3 задачі

3. Фільтрує тільки активні

5. Перевіряє фінальний стан

**Підказки:**
- Використовуйте ланцюжки команд
