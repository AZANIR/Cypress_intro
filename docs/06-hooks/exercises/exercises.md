# Практичні вправи - Тема 6: Хуки та анотації

## Вправа 1: Використання beforeEach()

**Завдання:**
Створіть тест з `beforeEach()`, який:
1. Відкриває сторінку
2. Додає 2 тестові задачі
3. Використовує ці задачі в 3 різних тестах

**Підказки:**
- `beforeEach()` виконується перед кожним тестом
- Кожен тест має свій власний стан

**Рішення:** Дивіться в папці `solutions/`

---

## Вправа 2: Очищення з afterEach()

**Завдання:**
Створіть тест з `afterEach()`, який:
1. Видаляє всі задачі після кожного тесту
2. Перевіряє, що список порожній

**Підказки:**
- Використовуйте `afterEach()` для очищення
- Це забезпечує ізоляцію тестів

**Рішення:** Дивіться в папці `solutions/`

---

## Вправа 3: Використання анотацій

**Завдання:**
Створіть тести з:
1. `it.skip()` - пропущений тест
2. Тест з кастомним таймаутом
3. Звичайний тест

**Підказки:**
- Використовуйте `.skip()` для тимчасового вимкнення
- Використовуйте `{ timeout: ... }` для таймаутів

**Рішення:** Дивіться в папці `solutions/`
