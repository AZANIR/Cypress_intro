# Практичні вправи - Тема 7: Конфігурація

## Вправа 1: Налаштування baseUrl

**Завдання:**
1. Відкрийте `cypress.config.js`
2. Додайте `baseUrl` для TodoMVC
3. Створіть тест, який використовує `cy.visit('/')`

**Підказки:**
- baseUrl: `'https://todomvc.com/examples/react/dist/'`
- Після налаштування можна використовувати відносні шляхи

**Рішення:** Перевірте `cypress.config.js` в папці `examples/`

---

## Вправа 2: Налаштування viewport

**Завдання:**
Створіть тест, який:
1. Тестує на desktop розмірі (1280x720)
2. Тестує на tablet розмірі (768x1024)
3. Тестує на mobile розмірі (375x667)

**Підказки:**
- Використовуйте `cy.viewport(width, height)`
- Або використовуйте пресети: `cy.viewport('iphone-x')`

**Рішення:** Дивіться в папці `solutions/`

---

## Вправа 3: Налаштування таймаутів

**Завдання:**
Створіть тест з:
1. Кастомним таймаутом для окремої команди
2. Кастомним таймаутом для всього тесту

**Підказки:**
- `{ timeout: 10000 }` для команди
- Третій параметр `it()` для всього тесту

**Рішення:** Дивіться в папці `solutions/`
